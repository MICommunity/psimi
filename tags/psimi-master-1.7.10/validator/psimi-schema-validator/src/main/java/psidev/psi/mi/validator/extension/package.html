This package contains a specific extension of the Validator framework for the PSI-MI 2.5 Schema.
Default validators classes have been specialized to fit the MI 2.5 needs. Pachage structure: 
    model: classes autogenerated by JAXB (these classes should not be edited manually)
    rules: the distinct rules implemented for that specific validator.
    test:  eventual tests implemented for that validator.
             we encourage people to write JUnit test to check the good behaviour of their rule.
    xpath: Helper classes that simplify the manual handling of the Context of an Error.
